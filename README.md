# Case Study II: The Redevelopment of Yue Man Square in Kwun Tong

Part of the project **Barriers to Housing Supply: Four Case Studies of Hong Kong**.

This case study examines the urban renewal process surrounding Yue Man Square in Kwun Tong, one of the most significant redevelopment projects undertaken by the Urban Renewal Authority (URA) in Hong Kong.
